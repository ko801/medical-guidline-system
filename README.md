A multi-page medical website built using *HTML, CSS, and JavaScript*.  
Includes a doctor chat service, login page, and medicine shopping page.

- *Home Page* – Introduction and navigation  
- *Services Page* – Patient ↔ Doctor chat UI  
- *Login Page* – Basic login form with validation  
- *Medicine Page* – Buy medicines with images  

- Responsive multi-page design  
- Clean UI and easy navigation  
- Doctor chat interface (frontend only)  
- Login form validation (JS)  
- Medicine grid with images  
- Built using pure HTML + CSS + JavaScript  

- *HTML5*  
- *CSS3*  
- *JavaScript*
